# Building Status

A Next.js application for managing building status information.

## Tech Stack
- Next.js 14 with App Router
- TypeScript
- PostgreSQL with Drizzle ORM
- Vitest for testing

## Commands
- `npm test` - Run tests
- `npm run build` - Production build
- `npm run dev` - Development server

## Pre-PR Requirements
- Always run `npm test` and ensure all tests pass before creating a PR
- Always run `npm run build` and ensure it succeeds before creating a PR
